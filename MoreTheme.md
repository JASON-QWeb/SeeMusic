新主题实施计划
1. 需求概述
添加两个新的可视化主题，这两个主题的 UI 窗口都是正方形（与现有的横向波浪窗口不同）。

2. 主题设计
🟢 主题一：绿色音响柱状图 (Equalizer)
视觉效果：

5 列垂直柱状图，每列由多个方格堆叠组成
颜色：绿色渐变（底部深绿 → 顶部亮绿）
每列的高度根据音量实时变化
各列高度不同，形成整体起伏波动效果
交互逻辑：

rms 控制整体柱高
lowEnergy 控制左侧两列（低频）
高频能量控制右侧两列
中间列取平均值
窗口尺寸：200×200 正方形

🟢 主题二：3D 粒子脉冲圆面 (Particle Pulse)
视觉效果：

椭圆/圆形区域，由绿色粒子填充
粒子具有 3D 深度感（大小/透明度表示远近）
音量越大，中心粒子越"靠近"（变大变亮）
整体呈现呼吸/脉动效果
交互逻辑：

rms 控制粒子整体"膨胀"程度
beat 触发粒子颤动/震动效果
climaxLevel 控制粒子密度和亮度
窗口尺寸：200×200 正方形

3. 技术方案
3.1 代码变更
[MODIFY] 
Config.swift
添加两个新的 Theme 枚举值：
equalizer = "音响"
particle = "脉冲"
添加窗口尺寸自适应逻辑（正方形主题使用不同尺寸）
[NEW] 
EqualizerView.swift
绿色音响柱状图主题的实现：

5 列 VStack 堆叠的方块
使用 Canvas 或 Path 绘制
响应 AudioCaptureService.shared.currentFeatures
[NEW] 
ParticlePulseView.swift
3D 粒子脉冲圆面主题的实现：

使用 Canvas 绘制多个粒子点
每个粒子有：位置、大小、透明度、深度
音量驱动粒子的 scale 和 offset 动画
[MODIFY] 
ContentView.swift
根据当前主题切换不同的视图：
classic/minimal/neon → WaveView()
equalizer → EqualizerView()
particle → ParticlePulseView()
根据主题调整窗口尺寸
[MODIFY] 
FloatingPanel.swift
支持正方形窗口尺寸
主题切换时自动调整窗口大小
3.2 窗口尺寸策略
主题类型	窗口尺寸
波浪类（蓝白/简约/霓虹）	480×140（横向）
音响柱状图	200×200（正方形）
粒子脉冲	200×200（正方形）
