# START

## 开发环境要求
- macOS 14.2+
- Xcode 15.2+（或同代版本）
- 可用的系统音频输出设备（扬声器/耳机）

## 启动步骤
1. 用 Xcode 打开 `SeeMusic.xcodeproj`。
2. 选择运行设备为 `My Mac`。
3. 选择运行 target 为 `SeeMusic`。
4. 点击 Run（⌘R）启动。
5. 若出现权限提示，按提示授权并重新运行。

## 第一次运行需要的权限
- 麦克风权限（用于音频采集/可视化）
- 系统音频捕获/屏幕录制权限（若系统提示）
- 辅助功能权限（若系统提示）

## 如何验证启动成功
- 桌面出现透明置顶悬浮窗，波浪持续动画。
- 播放音乐时波浪起伏增强；静音时幅度回落但仍有轻微呼吸。

## 常见启动失败原因
- 未授予麦克风/系统音频/辅助功能权限。
- 运行目标选择错误（未选 `SeeMusic` 或未在 `My Mac` 上运行）。
- 系统无音频输出或设备静音导致波浪不动。
- macOS 或 Xcode 版本过低。
