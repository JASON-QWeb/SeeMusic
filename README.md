# SeeMusic （For Mac）

单纯听歌也太无聊了！跟着节奏律动！

## Mac 音乐麦浪悬浮窗

- 无论你在用什么播放器听歌（网易云音乐、QQ音乐、AppleMusic、Spotify等），将节奏律动起来！
- 支持Mac版本：macOS 14.2+

## 使用指南

- 灵敏度：律动程度
- 主题：切换不同动画效果
- 帧率：调节帧率

## 技术说明

核心算法由“音频特征提取 + 平滑 + 非线性映射 + 节拍检测”驱动：
- 采集系统音频，提取 RMS、人声能量、低频差分节拍、高潮程度等特征。
- 使用噪声地板抑制底噪，配合 gamma 压缩与 Attack/Release 平滑，保证小音量可见且不抖动。
- 波浪主题将特征映射到振幅/尖锐度并加节拍脉冲；均衡器主题把节拍、低频和 RMS 混合为 5 列能量并独立平滑；鼓面主题用“膜面波形 + 冲击环 + 边缘阻尼”模拟鼓面震动。
